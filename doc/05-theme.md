# 主题

设置主页主题：ADMIN后台 -> COMMON菜单 -> indexTheme -> 提交

设置搜索页主题：ADMIN后台 -> COMMON菜单 -> searchTheme -> 提交

设置文章列表页主题：ADMIN后台 -> CATEGORY菜单 -> 左侧:点击文件树中的文件夹 ->
    右侧：点击"ADVANCED SETTING" -> 选择主题 -> 提交

设置文章列表页主题：ADMIN后台 -> CATEGORY菜单 -> 左侧:点击文件树中的文件 ->
    右侧：点击"ADVANCED SETTING" -> 选择主题 -> 提交
